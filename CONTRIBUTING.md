# How to Contribute to Constellinux

- Adding A Module
	- You can add a module at [Needs Adding]
	- Modules are apps ran in the *Rask* Terminal
		- eg: 
			- ```module test```
			- ```module returned "Hello"```
- Adding An App
	- There is currently no way to do this (I expect to add this for *BootSystem 1.0*, which will be an overhaul.)
- Finding Bugs
	- Finding bugs is an easy way to contribute, all you have to do it find a bug (pretty self explanitory, huh) and then add your name to ```bug-finders.md```.
