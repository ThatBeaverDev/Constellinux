Constellinux Uses the <a href="https://homepages.uc.edu/~thomam/Intro_Unix_Text/File_System.html">Unix Filesystem</a> 

The Default File Structure Looks Like This:

- /
	- root/
	- home/
		- root/
			- desktop/
			- documents/
			- downloads/
			- media/
				- videos/
				- photos/
				- music/

/Home/ contains the user directories, like `C:\Users` in windows and `/Users/` in macOS.
/Home/ will also include the folder of any other added users.
